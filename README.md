# 🍽️ Smart QR-Based Restaurant Ordering System

## 📌 Overview

A full-stack intelligent restaurant system that allows customers to scan QR codes, place orders, and receive real-time updates, integrated with AI-based recommendation and analytics.

---

## 🚀 Features

* 📱 QR-based menu access
* 🛒 Contactless food ordering
* 🔄 Real-time order tracking (Socket)
* 🤖 AI-based food recommendation
* 😊 Sentiment analysis on feedback
* 👨‍🍳 Kitchen dashboard for live orders

---

## 🧠 AI Capabilities

* Recommendation system
* Sentiment analysis
* Order prediction logic

---

## 🛠️ Tech Stack

**Frontend:**

* React.js

**Backend:**

* Node.js (Express)

**AI Service:**

* Python (FastAPI / ML models)

**Database:**

* PostgreSQL / MongoDB (based on your config)

---

## ⚙️ Setup Instructions

### 1. Clone the repo

```bash
git clone https://github.com/ismail-sha/restaurant-qr-system.git
cd restaurant-qr-system
```

### 2. Backend setup

```bash
cd backend
npm install
npm start
```

### 3. AI Service

```bash
cd ai-service
pip install -r requirements.txt
python main.py
```

### 4. Frontend

```bash
cd customer-app
npm install
npm start
```
